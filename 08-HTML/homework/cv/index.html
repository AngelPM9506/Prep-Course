Experiencia escolar

Proyectos de investigación y Titulación

Modelado y simulación de un nanocompósito de nanopartículas cerámicas en matriz polimérica

Tipo de Proyecto: Modelado y Simulado

Función: Predecir el comportamiento eléctrico del nanocompósito

Descripción:

Durante el desarrollo de este proyecto se realizo un modelo tridimensión de nano partículas de CaO · SiO2, esféricas con un radio de 1 – 200nm, suspendidas en una Poli(dimetil siloxano), en el programa MathLab, para después simular los efectos de un arco corona en el material modelo con el programa ComSol, con lo cual se demostró teóricamente que este tipo de material tiene una resistencia mayor a la degradación por calentamiento y efectos de sobrecarga eléctrica ante el evento del arco corona.

Tecnologias: MatLab, ComSol

Diseño y desarrollo de un molino de bolas automatizado para laboratorio de nanotecnología

Tipo de Proyecto: Diseño y automatización

Función: Obtención de nanopartículas a través de una metodología de top-down.

Descripción:

Durante el desarrollo de este proyecto, se diseño y desarrollo un Molino de bolas desde cero, totalmente automatizado para el laboratorio de nano tecnología, con la finalidad de determinar la factibilidad de obtener nanopartículas mediante el método de molienda de cualquier material rígido, el diseño tridimensional, el sistema de control de potencia y programación es de total autoría propia. Se logro determinar la posibilidad de obtener partículas por debajo de una 1um, pero como no se alcanzaron tamaños de partículas menores a 0.1um, por lo que no se podían considerar nanomateriales.

Tecnologias: AutoCAD, Arduino, Excel, ComSol, Visual Studio

Control ambiental para cuarto de estudio

Tipo de Proyecto: Aplicación de Io

Función: Controlar clima y luminosidad en un cuarto de estudios

Descripción:

Mediante la ultimación del micro controlador arduino se realizo un sistema de control ambiental para un cuarto de estudio, se controlo temperatura, luminosidad gradual, encender y apagar una lampara de escritorio, a partir de uno o varios dispositivos conectados en una red privada (wifi o bluetooth).

Tecnologias: App Inventor, arduino

Plataforma web para gestión educativa en preparatorias con agrupación académica mediante una Red Neuronal Artificial

Tipo de Proyecto: Aplicación web y redes neuronales

Función: Aplicación para la gestión académica total de una preparatoria y agrupación académica más eficiente mediante una RNA.

Descripción:

La finalidad de este proyecto era realizar una gestión completa una preparatoria a través de una única aplicación web, escrita en php y lograr una agrupación académica de estudiantes y profesores a fines afines en metodología de estudio y cuestiones con conductuales, esto mediante una red neuronal que evaluara y agrupara a cada estudiante y profesor de acuerdo a sus aptitudes y características, utilizando mismo php para la programación de la red neuronal, pero php demostró no ser la mejor opción para este tipo de paradigmas de programación.
